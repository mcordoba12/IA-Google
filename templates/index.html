¿Qué hace este sistema?

El clasificador utiliza un modelo Vision Transformer (ViT) para procesar imágenes dividiéndolas en parches y aplicando mecanismos de atención. El modelo fue entrenado con un gran conjunto de datos (ImageNet) y puede identificar categorías generales con distintos niveles de confianza.
